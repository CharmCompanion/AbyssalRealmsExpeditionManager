# Tales of Symphonia: Dawn of the New World - Monster, Item, Encounter, and Evolution Charts

This directory contains structured markdown tables for all monsters, items, encounters, and evolutions, grouped logically by family/type/area. Each chart omits redundant columns and is designed for clarity and reference.

- Each monster family has its own chart.
- Each item type has its own chart.
- Each area has its own encounter chart.
- Each evolution family has its own chart.

See the mechanics document for step-by-step gameplay systems and rules.
