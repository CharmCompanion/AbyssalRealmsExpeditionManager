## ISOMETRIC CAMERA SETUP & RENDERING WORKFLOW

### STEP 1: Install Pixel Art Addon
- Download from: https://lucasroedel.gumroad.com/l/pixel_art
- Install in Blender: Edit > Preferences > Add-ons > Install
- Enable the addon after installation

### STEP 2: Create Isometric Camera Rig
**Base Camera Settings:**
- Type: Orthographic (not Perspective)
- Orthographic Scale: Adjust based on building size (start with 5-10)
- Rotation:
  - X: 60Â° (for standard isometric) or 54.736Â° (for true isometric)
  - Y: 0Â°
  - Z: 45Â° (base angle)

**Camera Positioning:**
- Position camera far enough from buildings to fit in frame
- Center building at world origin (0, 0, 0) for easy rotation
- Adjust orthographic scale to frame building properly

### STEP 3: Setup 4-Direction Rotation Cameras
Create 4 cameras for rotation angles (or rotate one camera 4 times):

**Camera 1 - Front Right:** Z = 45Â°
**Camera 2 - Back Right:** Z = 135Â°
**Camera 3 - Back Left:** Z = 225Â°
**Camera 4 - Front Left:** Z = 315Â°

Keep X and Y rotation constant for all cameras.

### STEP 4: Configure Pixel Art Render Settings
**Output Properties:**
- Resolution: 512x512 (4x your final pixel size)
- File Format: PNG
- Color: RGBA (enable transparency)

**Render Engine (Eevee):**
- Enable: Ambient Occlusion
- Enable: Screen Space Reflections (if needed)
- Disable: Motion Blur
- Disable: Bloom (unless desired effect)

**Pixel Art Addon Settings:**
- Pixel size: 4 (renders at 4x then downscales)
- Use nearest neighbor filtering
- Enable texture filtering overrides

**World Settings:**
- Set transparent background: Film > Transparent

### STEP 5: Batch Render All Angles
**Option A - Manual:**
1. Select Camera 1, render, save as "building_name_0.png"
2. Select Camera 2, render, save as "building_name_90.png"
3. Select Camera 3, render, save as "building_name_180.png"
4. Select Camera 4, render, save as "building_name_270.png"

**Option B - Python Script:**
```python
import bpy

building_name = "cottage"
angles = [45, 135, 225, 315]

for i, cam_name in enumerate(cameras):
    bpy.context.scene.camera = bpy.data.objects[cam_name]
```
