# Blueprint & Template System

This module will handle saving, loading, and placing building blueprints and templates.

## Features
- Save buildings as blueprints (part list, transforms, metadata)
- Load and place blueprints
- Hologram/template guides for placement
- Quick build from blueprint list

## Key Scripts
- blueprint_resource.gd: Resource for blueprint data
- blueprint_manager.gd: Handles saving/loading/placement
