# Modular Building System

This module will handle part-by-part building construction, grid snapping, and advanced placement features.

## Features
- Modular part placement (walls, floors, roofs, etc.)
- Grid snapping and hologram preview
- Crafting/buying parts
- Advanced transforms: rotate, skew, tilt, resize, flip, mirror

## Key Scripts
- modular_building.gd: Main logic for modular construction
- part_resource.gd: Resource definition for building parts
