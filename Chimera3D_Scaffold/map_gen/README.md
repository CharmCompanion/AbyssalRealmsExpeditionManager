# Map Generation System

This module will handle procedural 3D map generation, kingdom/biome selection, and seed-based variation.

## Features
- Kingdom selection and biome assignment
- Procedural terrain generation (noise-based)
- Seed-based world variation
- Biome-specific asset placement

## Key Scripts
- map_gen.gd: Main entry point for map generation
- biome_data.gd: Biome and kingdom data definitions
